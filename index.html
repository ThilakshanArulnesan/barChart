This stacked barchart has been generated using the parameters:

  • height: 500
  • width: 500
  • color: ["#7293cb","#e1974c"]
  • titleText: "Demo stacked barchart"
  • titleColor: "Blue"
  • titleSize: "30"
  • spacing: "50"
  • labelCentering: "center"
  • labelColor: ["red", "white"]
  • numTicks: 7
  • yMax: 7

With the given data [ [1, 2], [2, 3], [3, 1]]. Note that the label colors and the bar colors can be changed individually (an array the same size as the stacked data must be used). Note that the width of the bars depends on the spacing between the bars as well as the width of the barchart overall.
